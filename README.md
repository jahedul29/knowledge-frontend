## knowledge-frontend

### Link : [https://knoledge-store.netlify.app/](https://knoledge-store.netlify.app/)

### Features

- Register user
- Login user
- Add book by authenticated user
- Edit and delete book by author
- Searchable and filterable Book list
- Book details
- Wishlist (Reading or Read later)
